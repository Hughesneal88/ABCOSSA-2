# ABCOSSA

Website for the Animal Biology and Conservation Science Student Association (ABCOSSA) at the University of Ghana.

## Tech stack

- Vite + React
- TypeScript
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd abcossa-connect

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Building

```sh
npm run build
```

The production build outputs to the `docs/` folder (used for GitHub Pages deployment).
